# Clush_Assignment_Project

## 프로젝트 소개 및 구현기능

Clush 기업 과제입니다. 할일의 생성,표시,수정,삭제가 가능한 Todo앱입니다. 수정기능에는 두가지가 있는데, 체크박스를 클릭하여 할일을 완료했다는 기능과 수정버튼을 눌러 할 일의 내용을 수정하는 기능이 있습니다.
